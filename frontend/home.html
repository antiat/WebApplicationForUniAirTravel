О проекте

Веб-приложение автоматизирует основные процессы авиакомпании: управление рейсами, регистрацию пассажиров, бронирование билетов и администрирование системы. Реализованы ролевая модель доступа (RBAC), JWT-аутентификация и полноценный REST API.

Технологии Python 3.11+ Django 4.2 Django REST Framework JWT Auth PostgreSQL Bootstrap 5 Swagger / OpenAPI
